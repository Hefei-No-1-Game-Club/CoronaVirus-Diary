User FAQ

My game does not start, please help me!

Make Sure you have downloaded the correct installer and the installer is not damaged. Usually this does not happen, but it may occur when the network connection is not stable.

I am installing the game on Windows, and the installer says the developer is not trusted.

The Windows Installer is signed with our own certificate, so it may experience a warning. Please download the certificate here and follow the instructions from Microsoft to install the certificate. Then, the installer will work.

I successfully installed the game on my mac, but the game cannot open.

The mac game is also not properly signed. Please download this scriplet to repair your application.



Developer FAQ

The Unity Editor says the build setting is not supported.

Yes, sometimes we may generate the build on mac or linux and then directly submit it to GitHub. You may experience this error when you are opening the project on a machine does not contain those two building platform tools. Simply click on Continue and ignore it.
